# DevTracker

DevTracker is a lightweight task tracking web app built with React and Vite.

## Features
- Task groups and dashboards
- Dark/light theme support
- Local storage persistence

## Run locally

Install dependencies:

```bash
npm install
```

Start dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

## Deploy on Vercel
1. Push repository to GitHub.
2. Import project to Vercel and set framework to `Vite`.
3. Build command: `npm run build`, Output directory: `dist`.
